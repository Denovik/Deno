# Fähigkeit: Daten

Dein Mitarbeiter bekommt echte Zahlen aus deinen Plattformen — automatisch, ohne dass du sie von Hand einträgst.

## Was diese Fähigkeit tut

- Holt täglich aktuelle Zahlen von YouTube, Instagram und TikTok
- Schreibt sie automatisch in `context/current-data.md`
- Damit kennt dein Mitarbeiter beim nächsten `/prime` deinen echten Stand

## Was du dafür brauchst

- YouTube: läuft bereits (OAuth-Token vorhanden)
- Instagram: läuft bereits (Meta Access Token vorhanden)
- TikTok: nach Review-Freigabe

## Einrichten

```
/install module-installs/daten
```
